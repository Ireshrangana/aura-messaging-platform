# AI Integration Points

## End-user AI

- Smart replies above the composer
- Thread summaries and unread catch-up cards
- Writing tools: rewrite, shorten, expand, fix grammar, translate
- Voice note transcription and summarization
- Semantic search by topic, file, person, or intent

## Safety AI

- Link and spam risk classification
- Group moderation suggestions
- Abuse triage queue with confidence scoring

## Guardrails

- Provider abstraction to avoid lock-in
- Prompt templates per capability
- Message and file access filtered by permissions
- AI actions stored in `ai_usage_logs`
- Admin-only visibility into aggregate AI analytics
